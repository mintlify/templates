## General

| Action | Shortcut |
|--------|----------|
| Command palette | ⌘K / Ctrl+K |
| Search | ⌘P / Ctrl+P |

## Editor

| Action | Shortcut |
|--------|----------|
| Bold | ⌘B / Ctrl+B |
| Comment | ⌘⌥M / Ctrl+Alt+M |

## Customize

Some shortcuts can be remapped in **Settings → Accessibility** on supported clients.
