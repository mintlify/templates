## Goal

Route new items to a reviewer and notify Slack on approval.

## Steps

1. Create a workflow from a template.
2. Add a human approval step with timeout.
3. Connect Slack notification on success.
4. Test in a sandbox project.

## Validation

Run dry-run executions with sample payloads before enabling for all projects.
